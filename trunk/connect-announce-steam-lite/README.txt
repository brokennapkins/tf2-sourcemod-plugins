'Connect Announce Steam Lite' for displaying players' Steam IDs

For displaying an alternate join message to admins showing a player's Steam ID

All messages printed to the text chat area

CVARs:
sm_connect_announce_steam_lite_enable <0|1> - 0 (default) turns off this plugin, 1 turns on this plugin
sm_connect_announce_steam_lite_version - returns the current version of this plugin

Based off of the 'Connect Announce' plugin with all the fat trimmed.
